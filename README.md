# solder

Description:
I made a little cat which has a photoresistor to turn on two leds when dark, and also has two leds which turn on by button.

BOM:
- 4x LED
- 5x Resistor
- 1x Photoresistor
- 1x NPM Transistor
- 1x Battery
- 2x Button
